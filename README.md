# YT Lyrics

YouTube でリアルタイム歌詞を表示する Chrome 拡張機能。lrclib.net の同期歌詞 (LRC) を YouTube 動画の右側パネルに表示します。

![status](https://img.shields.io/badge/manifest-v3-blue) ![license](https://img.shields.io/badge/license-MIT-green)

## 主な機能

- 🎵 **同期歌詞表示** — lrclib.net から LRC 形式の歌詞を取得して、再生位置に合わせて自動スクロール
- 🎯 **タップ同期** — 動画の任意の位置で歌詞行をクリック → そのタイミングを基準にオフセット自動計算
- 🔍 **手動検索** — タイトル誤検出時に曲名で直接検索して差し替え
- ⏱ **自動オフセット補正** — 動画と歌詞の長さの差異・字幕テキストの相関から自動調整
- 🎌 **フリガナ表示** — kuromoji.js による日本語形態素解析で漢字の上にフリガナ表示
- 📺 **フォーカスモード** — 歌詞を大きく表示するレイアウトに切り替え
- ⚡ **プレイリスト先読み** — 次の曲の歌詞を事前取得して即座に表示

## インストール

1. このリポジトリを clone
   ```
   git clone https://github.com/harisen/yt-lyrics.git
   ```
2. Chrome で `chrome://extensions/` を開く
3. 右上の「デベロッパーモード」を ON
4. 「パッケージ化されていない拡張機能を読み込む」をクリック
5. clone したフォルダを選択

## 使い方

1. YouTube で音楽動画を開く
2. 右サイドバーに歌詞パネルが自動表示される
3. 歌詞がずれている場合:
   - オフセットバーの ±1 / ±0.1 ボタンで微調整
   - または 🎯 ボタン → 合わせたい位置で再生 → 該当する歌詞行をクリック（タップ同期）
4. 曲が誤判定された場合:
   - 🔍 ボタン → 曲名で検索 → 正しい曲を選択

## オプション機能

| ボタン | 機能 |
|---|---|
| ⤢ | フォーカスモード (歌詞拡大) |
| ↻ | 歌詞を再読み込み |
| ふ | フリガナ ON/OFF |
| A− / A＋ | フォントサイズ調整 |
| 🔍 | 手動検索バー開閉 |
| 🎯 | タップ同期モード |
| ✕ | パネルを閉じる |

## データ・プライバシー

- 歌詞は [lrclib.net](https://lrclib.net) （オープンソースの歌詞コミュニティ API）から都度取得します
- 歌詞データは **保存・再配布しません**（セッション中のキャッシュのみ）
- オフセット調整値のみ `localStorage` に動画 ID 単位で保存します
- 外部に個人情報を送信することはありません
- 認証・ログインは不要です
